playlister
about

Playlister is a webapp implementing the Twilio API, MongoDB, Youtube Search API, and Google Custom Search API. It was developed at HackRU Fall 2015. Playlister acts as an on the go service by allowing users to add their favourite music through the SMS using the name of the song itself or searching for songs matching the given lyrics.

technologies used

Youtube Data API v3 - used to parse for Youtube searches when adding songs
Google Custom Search API - used to search for matching lyrics
NodeJS and Express - backend management
MongoDB - storing users data
Twilio - Sending and receiving text messages
Linode - used to host the NodeJS server

usage

add [SONG_NAME]: adds the song to your playlist
lyric [SONG_LYRICS]: returns song name and adds the song to your playlist

When you want to view your playlist navigate to http://tinyurl.com/pllister and enter your phone number
