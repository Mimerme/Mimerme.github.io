Yak Hax API

What is it?

Yak Hax is a Java library that wraps the already existing Yik Yak API for third party developers. The current Yik Yak API remains closed off and remains unavalible for those interested in data minning, or managing third-party clients.

Technologies Used

Yik Yak does their networking through HTTPS, so I used Charles and my Android device to caputre the network traffic. I decompiled the Yik Yak official app and reverse engineered it to extract parts that I needed. I also used Fiddler to capture my computer's network traffic and used Eclipse to write the library itself. I also got help from other memebers of the Yik Yak Hacking community to create this lbrary and extract the necessary API keys
